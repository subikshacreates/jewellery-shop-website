# 💎 Product Requirement Document (PRD)
## Jewellery Shop Website (Anti-Tarnish Collection)

---

## 1. 📌 Overview
A web-based digital showroom for a jewellery shop to showcase anti-tarnish products and generate customer enquiries via WhatsApp.

---

## 2. 🎯 Objectives
- Build brand trust
- Showcase products professionally
- Convert visitors into enquiries
- Keep user flow simple (no login required)

---

## 3. 👥 Target Users
- Men and Women
- Age: 18–45
- Interested in affordable, stylish jewellery

---

## 4. 🛍️ Product Scope

### Core Features:
- Product catalog with categories
- Product details (name, price, image)
- WhatsApp enquiry system
- Admin panel (add/update products)
- Offers & discount section
- Testimonials & feedback form

---

## 5. ⚙️ Functional Requirements

### User Side:
- View products by category
- Click product → send WhatsApp enquiry
- View offers and testimonials
- Submit feedback

### Admin Side:
- Add / edit / delete products
- Manage enquiries
- Update offers

---

## 6. 🚫 Non-Functional Scope
- No online payment (initial version)
- No mandatory login system
- Fast loading and mobile responsive

---

## 7. 🔄 User Flow

1. User visits website
2. Browses categories
3. Selects product
4. Clicks "Enquire"
5. Redirected to WhatsApp

---

## 8. 🧠 Design Goals
- Premium look (black + gold theme)
- Clean UI
- Mobile-first design

---

## 9. 📦 Tech Stack
- Frontend: Next.js
- Backend: Firebase
- Database: Firestore
- Hosting: Vercel

---

## 10. 📈 Future Enhancements
- User login (optional)
- Online payments
- Order tracking
